# 초기 데이터 설정 가이드

## 자동 설정된 초기 데이터

페이지가 처음 로드될 때 다음 데이터가 자동으로 Google Sheets에 저장됩니다:

### 1. 스트레칭 영상
- **URL**: `https://youtu.be/oYmiL6Bh_ak`
- **제목**: 구영테크 스트레칭 영상
- **설명**: 근골격계 부담 예방을 위한 실전 스트레칭 가이드

### 2. 의약품 설문조사 링크
- **URL**: `https://docs.google.com/forms/d/e/1FAIpQLSc8GB-tbu8QhWnoItYi7lS26tyLX0KzQvNZ-bwc-Cqw-bHpsQ/viewform?usp=sharing&ouid=110501763882946908099`

### 3. 배치도면 이미지
- 배치도면은 관리자 모드에서 직접 업로드해야 합니다.
- **방법**:
  1. 관리자 모드로 로그인
  2. "작업환경측정 - 소음 현황" 탭으로 이동
  3. "1공장 이미지 추가" 또는 "2공장 이미지 추가" 버튼 클릭
  4. 이미지 파일 선택 (PNG, JPG 등)
  5. "저장" 버튼 클릭

## 수동 설정 방법

### Google Sheets에서 직접 설정

1. **Google Sheets 열기**
   - Google Apps Script와 연결된 스프레드시트 열기

2. **Config 시트 열기**
   - "Config" 탭 클릭

3. **데이터 입력**

   **스트레칭 영상 (stretchingVideos 키):**
   ```
   key: stretchingVideos
   value: [{"id":1,"title":"구영테크 스트레칭 영상","description":"근골격계 부담 예방을 위한 실전 스트레칭 가이드","videoUrl":"https://youtu.be/oYmiL6Bh_ak","thumbnail":"","duration":"","category":"전체","createdAt":"2026-02-12"}]
   ```

   **의약품 설문조사 URL (medicine_survey_url 키):**
   ```
   key: medicine_survey_url
   value: https://docs.google.com/forms/d/e/1FAIpQLSc8GB-tbu8QhWnoItYi7lS26tyLX0KzQvNZ-bwc-Cqw-bHpsQ/viewform?usp=sharing&ouid=110501763882946908099
   ```

   **배치도면 (floorplans 키):**
   - 배치도면은 base64 인코딩된 이미지 데이터를 저장해야 합니다.
   - 관리자 모드에서 업로드하는 것이 가장 쉽습니다.

## 확인 방법

1. **브라우저 콘솔 확인**
   - F12 키를 눌러 개발자 도구 열기
   - 콘솔 탭에서 다음 메시지 확인:
     - `✅ 스트레칭 영상 초기값 저장`
     - `✅ 의약품 설문조사 URL 초기값 저장`
     - `✅ 초기 데이터 저장 완료`

2. **페이지 새로고침**
   - 데이터가 저장된 후 페이지를 새로고침하면 자동으로 표시됩니다.

3. **다른 기기에서 확인**
   - 다른 컴퓨터나 핸드폰에서 GitHub Pages URL로 접속
   - 최대 10초 내에 데이터가 자동으로 표시됩니다.

## 주의사항

- 초기 데이터는 **한 번만** 자동으로 저장됩니다.
- 이미 데이터가 있으면 덮어쓰지 않습니다.
- 배치도면은 관리자 모드에서 직접 업로드해야 합니다.
- 초기화를 다시 실행하려면 브라우저의 localStorage에서 `data_initialized` 키를 삭제하세요.
