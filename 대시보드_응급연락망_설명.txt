■ 금월 대상자 명단 / 응급연락망 메뉴얼 복구 안내

[원인]
구글 시트 Config 시트가 비어 있으면, 서버가 factory1·factory2·emergencyContacts를
빈 배열([])로 보냅니다. 그걸 그대로 덮어쓰면서 화면에 있던 "금월 건강상담 대상자 명단"과
"응급연락망"이 비어 보였습니다.

[수정 내용]
• index.html의 applyDataFromServer() 수정
  - 서버에서 보낸 Config 값이 "빈 배열"일 때는 덮어쓰지 않도록 변경
  - emergencyContacts, hospitals, factory1, factory2: 길이가 1 이상일 때만 덮어씀
  → Config를 비워 두면, index.html에 들어 있는 기본 명단·연락망이 그대로 표시됩니다.

[지금 하실 일]
1. 수정된 index.html 파일을 저장했는지 확인
2. G-System 페이지(index.html)를 새로고침(F5)
3. 대시보드에서 "금월 건강상담 대상자 명단"이 다시 보이는지 확인
4. 좌측 메뉴에서 "응급연락망" 탭 클릭 → "🚨 응급 연락망" 카드에 연락처가 다시 보이는지 확인

[나중에 Config 시트에 직접 넣고 싶을 때]
• 금월 대상자: Config 시트에 key = factory1, factory2 / value = JSON 배열
  예) key: factory1  value: ["홍길동","김철수","이영희"]
• 응급연락망: Config 시트에 key = emergencyContacts / value = JSON 배열
  (각 항목에 id, category, name, phone, description, icon, color, order 등)
구글 시트에 이렇게 채워 넣고 저장하면, 다음 새로고침부터 시트 데이터가 우선 적용됩니다.

[정리]
- Config 비어 있음 → 화면에는 index.html 기본값(금월 명단·응급연락망) 표시
- Config에 데이터 있음 → 시트 데이터로 표시
