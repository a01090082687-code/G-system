## 이번 확정에 포함된 수정 요약

1. **index.html**  
   - 오락실 버튼 3개만 표시 (🔫 면역 전쟁, 🛡️ 보건 서바이벌, 🏃 헬스 런)  
   - 클릭 시 `location.href`로 각각 `immunity-war.html`, `health-survival.html`, `health-run.html` 이동  

2. **health-survival.html (보건 서바이벌)**  
   - 게임 시작 버튼/화면 터치 시 확실히 시작되도록 수정  
   - `startGame()` 전면 검토: 캔버스·ctx 보호, animationId 확정, 이벤트 중복 방지  
   - 버튼 클릭 시 `state.run = true` 후 게임 루프가 한 번은 반드시 스케줄되도록 처리  

3. **health-run.html (헬스 런)**  
   - 스크롤만으로 이동하도록 방향 정리 (장애물/아이템 월드 x 이중 이동 제거)  
   - 구펭 오른쪽 보기: 그리기 시 `scale(-1,1)` 적용  
   - 점프 곡선 쿠키런 스타일: 상승/하강 중력 분리 및 점프력 조정
